Elevation vs Pace Terrain

How much does climbing slow you down? 1,685 runs with elevation data.

Vertical Gain per Mile vs Pace

Each dot is a run. X = feet of climbing per mile. Y = pace. The cost of going uphill.
Every 100 ft/mi of climbing costs roughly 1:30/mi of pace. Flat runs (0-20 ft/mi) average ~8:00/mi. Moderate mountain runs (100 ft/mi) average ~9:30/mi. Steep mountain runs (200+ ft/mi) push past 11:00/mi. The relationship is remarkably linear up to about 150 ft/mi, then the curve steepens.

The Comeback Curve Resilience

After taking time off, how does the first run back compare to pre-break pace?

Pace Change After Rest

Difference in pace (seconds) for the first run back vs average of 5 runs before the break. Negative = faster.

Cloud Cover Effect Sky

Pace by Cloud Coverage

Clear skies (0-20%) produce the fastest runs at 8:48/mi. Partly cloudy (20-40%) slows to 9:10 — a 22-second hit. But fully overcast (80-100%) is only 9:03 — faster than partly cloudy. Theory: clear mornings = cool starts. Partial cloud = midday heat. Overcast = cooler all day.
